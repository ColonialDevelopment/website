Colonial Room Reservations

Hi. This is where you can reserve rooms in Colonial.

This section is still under construction but basic functionality exists.
Let Nick (nyang@) know if anything is broken...
You can go to the following links:

Reserve a Room
View a reservation: /reservations/view/(id#)
View reservations in a given month: /reservations/(yyyy)/(mm)
View reservations in a given day: /reservations/(yyyy)/(mm)/(dd)

Note that only months/days with reservations have links - if for example 2016/04/20 has no reservations then /reservations/2016/04/20 returns 404 [for now!]

And of course feel free to poke around the admin site.

See /colonialsite/reservations/readme - reservations for more complete documentation
